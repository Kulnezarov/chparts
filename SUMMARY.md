### 3️⃣ Validation (Валидация)

#### `lib/checkoutValidation.ts`
```typescript
validateCheckout({
  name: "Иван",
  phoneDigits: "77001234567",
  deliveryType: "city",
  addressLine: "ул. Жагалева 123"
})
```

Проверяет:
- ✓ Имя (минимум 2 символа)
- ✓ Телефон (11 цифр, начинается с 7)
- ✓ Адрес (если доставка по городу)
- ✓ Город + Детали (если почта)

#### `lib/apiErrorHandler.ts`
```typescript
ApiErrorHandler.getErrorMessage(error, "default")
```

- 429 → "Слишком много попыток"
- 400 → "Проверьте данные"
- 500+ → "Ошибка сервера"
- Network → "Проверьте интернет"

### 5️⃣ Checkout Logic (Логика оформления)

```
Выбор доставки
    ↓
Если NOT pickup → только "Карта"
Если pickup → "Карта" ИЛИ "Наличные"
    ↓
Валидация полей
    ↓
Если ошибки → показать Alert
    ↓
Если OK → отправить на backend
    ↓
Если 429 → специальное сообщение
    ↓
На успех → Success экран
```

#### Event Handlers:
- Очистка ошибок при вводе
- Автоматический выбор оплаты
- Умное отключение кнопок
